# HUMAN-DEVELOPMENT-INDEX(HDI)

#Project Overview
This project is a Machine Learning-based web application that predicts the Human Development Index (HDI) category of a country using four development indicators. The application is built using Python, Flask, HTML, CSS, and a trained Machine Learning model.
#Features
Predicts the HDI category.
User-friendly web interface.
Flask-based backend.
Machine Learning model for prediction.
Displays prediction results instantly.

#Technologies Used
1.Python
2.Flask
3.Scikit-learn
4.NumPy
5.HTML
6.CSS

#Input Parameters
Life Expectancy
Mean Years of Schooling
Expected Years of Schooling
Gross National Income (GNI) Per Capita

#Output
The application predicts one of the following HDI categories:
Low
Medium
High
Very High
